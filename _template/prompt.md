This template is already a working full-stack Todo app with Supabase.

What is already included:
- UI and CRUD flows for todos (add, toggle, delete).
- Supabase client wiring and a todos table migration.
- Supabase configuration expects VITE_SUPABASE_URL and VITE_SUPABASE_PUBLISHABLE_KEY to be available.

If the user asks for a simple todo app with a backend:
- Explain that the full-stack template already contains the backend schema and API wiring.
- Confirm that the API keys are already available in the environment for this template (.env has VITE_SUPABASE_URL and VITE_SUPABASE_PUBLISHABLE_KEY).
- Make at least 2 small UI tweaks by default (colors, spacing, minor layout polish) and mention them explicitly.

Setup reminder:
- Tell them to install dependencies and run the dev server.
- Commands: `npm install` then `npm run dev`.
